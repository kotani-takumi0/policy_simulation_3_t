ご利用にあたって

入力された事業情報をもとに、過去の類似施策を検索し、検討資料の素案作成を支援します。初回利用時は OpenAI API キーや参照データの配置をご確認ください。

  • 「現状・目的」「事業名」「事業概要」は、内部向けの整理メモとして入力してください。
  • 分析結果は右側に表示され、必要に応じて JSON 形式で書き出すことができます。
  • 分析ログは自動で保存され、「分析ログ一覧」よりいつでも再確認・削除が可能です。
